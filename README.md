# 🚖 Uber Ride Data Analysis

## 📌 Project Overview

This project analyzes Uber ride data using Microsoft Excel to identify ride trends, booking patterns, peak hours, and customer behavior.

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Dashboard Design
* Data Cleaning

---

## 📊 Features

* Ride trend analysis
* Peak booking hour analysis
* Revenue insights
* Location-wise ride analysis
* Interactive dashboard
